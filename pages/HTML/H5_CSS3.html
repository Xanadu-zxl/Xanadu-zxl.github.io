# 结构标签

  • <article>标记定义一篇文章

# validity

  • customError 设置为 true, 如果设置了自定义的 validity 信息 setCustomValidity()
  • patternMismatch 设置为 true, 如果元素的值不匹配它的模式属性。pattern
  • rangeOverflow 设置为 true, 如果元素的值大于设置的最大值 max
  • rangeUnderflow 设置为 true, 如果元素的值小于它的最小值 min
  • stepMismatch 设置为 true, 如果元素的值不是按照规定的 step 属性设置
  • tooLong 设置为 true, 如果元素的值超过了 maxLength 属性设置的长度
  • typeMismatch 设置为 true, 如果元素的值不是预期相匹配的类型,如:email,url,Tel
  • valid 设置为 true，如果元素的值是合法的
  • valueMissing 设置为 true，如果元素 (required 属性) 没有值

  • 检测约束,没有满足任意约束返回false,所有符合则ture

# 音频,视频

# video

  • height: pixels 设置视频播放器的高度。
  • autoplay: autoplay 如果出现该属性，则视频在就绪后马上播放。
  • controls: controls 如果出现该属性，则向用户显示控件，比如播放按钮。
  • loop: loop 如果出现该属性，则当媒介文件完成播放后再次开始播放。
  • poster: URL 封面，直到用户点击播放按钮。
  • preload: auto|metadata|none 出现该属性，则视频在页面加载时进行加载，并预备播放。如使用 "autoplay"，则忽略该属性。
  • muted: muted 如果出现该属性，视频的音频输出为静音。
  • src: URL 要播放的视频的 URL。
  • width: pixels 设置视频播放器的宽度。

<source type="video/mp4">可以有多个;

# audio

<source src = "hh.mp3" type="audio/mpeg">可以有多个;

# video和audio API事件

  • loadedmetadata 当浏览器已加载音频/视频的元数据时触发。

  • canplay 当浏览器可以开始播放音频/视频时触发。

  • canplaythrough 当浏览器可在不因缓冲而停顿的情况下进行播放时触发。

  • play 当音频/视频已开始或不再暂停时触发。

  • pause 当音频/视频已暂停时触发。

  • progress 当浏览器正在下载音频/视频时触发。

  • timeupdate 当目前的播放位置已更改时触发。

  • volumechange 当音量已更改时触发。

  • seeking 当用户开始移动/跳跃到音频/视频中的新位置时触发。

  • waiting 当视频由于需要缓冲下一帧而停止时触发。

  • abort 当音频/视频的加载已放弃时触发。

  • durationchange 当音频/视频的时长已更改时触发。

  • emptied 当目前的播放列表为空时触发。

  • ended 当目前的播放列表已结束时触发。

  • error 当在音频/视频加载期间发生错误时触发。

  • loadeddata 当浏览器已加载音频/视频的当前帧时触发。

  • loadstart 当浏览器开始查找音频/视频时触发。

  • playing 当音频/视频在因缓冲而暂停或停止后已就绪时触发。

  • ratechange 当音频/视频的播放速度已更改时触发。

  • stalled 当浏览器尝试获取媒体数据，但数据不可用时触发。

  • seeked 当用户已移动/跳跃到音频/视频中的新位置时触发。

  • suspend 当浏览器刻意不获取媒体数据时触发。

# transform 转换

# CSS 2D转换

  • rotate()2D，在一个给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转
  • translate()平移，根据左(X轴)和顶部(Y轴)位置给定的参数，从当前元素位置移动;
  • scale()缩放，该元素增加或减少的大小，取决于宽度（X轴）和高度（Y轴）的参数
  • skew()倾斜,transform:skew(<angle> [,<angle>]);分别表示X轴和Y轴倾斜的角度，如果第二个参数为空，则默认为0;
  • skewX:正逆时针,负顺时针
  • skewY:负逆时针,正顺时针
  • matrix(a,b,c,d,tx,ty)矩阵matrix 方法有六个参数，包含旋转，缩放，移动（平移）和倾斜功能。

# CSS 3D转换

  • rotate3d(x,y,z,angle)
  • rotateX(angle)方法，围绕其在一个给定度数X轴旋转的元素。
  • rotateY(abgle)方法，围绕其在一个给定度数Y轴旋转的元素。
  • rotateZ(angle)方法，围绕其在一个给定度数Z轴旋转的元素。
  • scale3d(x,y,z) 定义3D缩放转换,不能省略
  • matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) 定义 3D 转换，使用 16 个值的 4x4 矩阵。

# 拓展属性

  • transform-style:flat|preserve-3d;指定嵌套元素在三维空间中的呈现方式

  • perspective:number|none;指定观察者与[z=0]平面的距离,使具有三位位置变换的元素产生透视效果

  • perspective-origin: x-axis y-axis; 指定透视点的位置,默认50%

  • backface-visibility:visible|hidden 属性定义当元素不面向屏幕时是否可见。

  • 如果在旋转元素不希望看到其背面时，该属性很有用。

# CSS3 多列

# Multi-column

  • columns:column-width 每列宽度||columns-count 列数 --优先考虑列数放不下就考虑宽度
  • column-width：auto|length；对象每列的宽度
  • column-count：<integer>|auto; 需要分割的列数
  • column-gap:normal 与字体大小相同;属性指定了列与列间的间隙
  • column-rule 属性是 column-rule-*:设置了列与列之间的边框的width宽度，style样式及color颜色
  • column-span:none|all; 指定元素要跨越多少列
  • column-fill:balance|auto-- balance列长短平衡,浏览器应尽量减少改变列的长度;auto列顺序填充，他们将有不同的长度
  • column-break-before:auto 自由判断|always 元素之前换行,产生新的列|avoid 不在元素之前换行

# 用户界面

  • text-overflow:clip,ellipsis 定义内联内容溢出时其块容器是否截断,结合white-space:nowrap;overflow:hidden;一起使用
  • outline:<outline-width>|<outline-style>|<outline-color>| offset外移
  • outline: 设置或检索对象外的线条轮廓,不占布局空间,画在border外,不影响元素尺寸
  • cursor 鼠标指针光标样式--pointer小手,progress加载
  • zoom:nprmal <number>|<percentage>检索或设置对象整个的缩放比,没有负值
  • box-sizing:centent-box 不包含padding和margin|border-box 包含padding和margin
  • resize:none不可以缩放,调节尺寸|both 可以调节宽高|horizontal 宽度|vertical 高度---原理是增加行内style,扩大元素区域
  • user-select:none是否允许选中文本||text|all|element
  • pointer-event:auto|none 何时成为属性事件的焦点target,鼠标点击事件是否触发

# 不常用属性

  • ime-mode:检索是否激活输入法;不用
  • column-fill:指定如何填充列 (浏览器不支持)
  • appearance:none|button|button-becel.... 改变外观样式 ,作用不变,都不兼容
