# 背景

  • background-color 包含内边距和边框,不包含外边距
  • image 包含内边距和边框,不包含外边距
  • position 起始位置,相对于容器,一个关键字则居中
  • attachment: fixed永远固定一定位置 scroll跟随滚动
  • repeat 是否重复
  • background 不分先后
