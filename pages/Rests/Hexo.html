# 环境配置:

  • 安装Git:地址

  • 安装node.js:地址

  • 然后选择一个文件夹 右键打开Git Bush 如图所示:

  • 输入命令 npm install hexo-cli -g全局安装Hexo;

  • 输入命令 hexo init 初始化这个文件夹;

- Hexo随后会自动在目标文件夹建立网站所需要的文件。安装依赖包 `npm install`; - - 执行之后会在目录中安装 node_modules - 输入命令 `hexo server`[info] Hexo is running at http://localhost:4000/. Press Ctrl+C to stop. - 表明Hexo Server已经启动了，在浏览器中打开 http://localhost:4000/， - 这时可以看到Hexo已为你生成了一篇blog你可以按Ctrl+C 停止Server ## 部署到Github - 现在页面有了准备部署到GitHub: - 注册Github账号，新建一个Github仓库： - 仓库名字必须为 `username.github.io`，username就是你的Github用户名 - 关联到Github账号： - 进入博客根目录，找到 _config.yml翻到最下面，修改为： 
deploy：
	type: git
	repo:https://github.com/username/username.github.io.git
	branch: master

  • 一定要有缩进,且冒号后面必须有一个空格!!

  • 然后执行： npm install hexo-deployer-git --save

  • 然后配置SSH:

  • SSH是一种安全协议，我们需要通过SSH将Git与Github进行绑定，才能通过Git向Github提交代码。

  • 生成公钥私钥ssh-keygen -t rsa -C "你的邮箱地址"

  • 你可以在C:\Users\username.ssh中找到。

  • 添加公钥到Github用记事本打开id_rsa.pub，将内容复制

  • 然后打开你的Github主页在右上角头像处——>Setting，进入个人设置界面

  • 进入Setting页面后点击SSH and GPG keys进入子页面，然后点击New SSH key按钮

  • 最后，将公钥id_rsa.pub的内容粘贴到Key处，点击Add SSH key。

  • 验证在git bash中输入：ssh -T git@github.com，等待提示

  • 为了稳定还可以通过git命令匹配:git config --global user.name "yourname" ,为自己设置的名字自己定义git config --global user.email "youremail" ,为自己的邮箱地址;

# Next主题下载:

在hexo文件夹中打开GitBush 输入: git clone https://github.com/iissnan/hexo-theme-next themes/next 克隆最新版next主题到本地themes文件夹里

然后修改，Hexo配置文件启用主题。 进入博客根目录找到_config.yml,找到theme：，将它改为theme: next即可

# 常用命令:

  • 清除生成的文件和缓存：hexo c
  • 生成静态页面：hexo generate //或 hexo g
  • 本地预览：hexo server //或 hexo s
  • 然后在浏览器中输入localhost：4000即可预览效果，Ctrl+C结束
  • 新建文章：hexo new post "title"
  • 新建页面：hexo new page "title"
  • 部署： hexo d
  • 等待片刻你可以在浏览器输入：username.github.io看看效果。

# 上传图片

  • 把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true
  • 在你的hexo目录下执行这样一句话npm install hexo-asset-image --save，这是下载安装一个可以上传本地图片的插件，来自dalao：dalao的git
  • 等待一小段时间后，再运行hexo n "xxxx"来生成md博文时，/source/_posts文件夹内除了xxxx.md文件还有一个同名的文件夹
  • 最后在xxxx.md中想引入图片时，先把图片复制到xxxx这个文件夹中，然后只需要在xxxx.md中按照markdown的格式引入图片：
  • ![你想输入的替代文字](xxxx/图片名.jpg)
  • 注意： xxxx是这个md文件的名字，也是同名文件夹的名字。只需要有文件夹名字即可，不需要有什么绝对路径。你想引入的图片就只需要放入xxxx这个文件夹内就好了，很像引用相对路径
