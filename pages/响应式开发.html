# 响应式网站概念

  • 弹性网格布局
  • 弹性图片
  • 媒体查询
  • 优点:
  1. 减少工作量
  2. 节省时间
  3. 针对每个设备会有正确的设计
  4. 搜索优化
  • 缺点:
  1. 加载更多的样式和脚本资源
  2. 设计比较难精确定位和控制
  3. 老版本浏览器兼容不好

# viewport属性

设备屏幕上可以用来显示网页的区域

  • width: 设置layout viewport的宽度
  • height: 设置layout viewport的宽度
  • initial-scale: 设置页面的初始缩放值
  • min(max)imum-scale:页面的最大最小缩放值
  • user-scalable:是否允许缩放,no/yes

# css计算单位

  • px
  • rem html中的font-size大小
  • em 继承父元素的字体大小
  • VH,VW viewport的百分比计量单位(1vw=1%width)
  • vmin,vmax 最大最小百分比
